How this works (so you can trust it, not us)
  1. We canonicalise the JSON exactly as it was signed: keys sorted, no whitespace, UTF-8 (json.dumps(sort_keys=True, separators=(",",":"), ensure_ascii=False)).
  2. Your browser's native WebCrypto Ed25519 implementation verifies the signature over those canonical bytes against the public key.
  3. Nothing is sent anywhere — open dev-tools → Network and watch: zero outbound calls during verification.
  4. The same scheme signs the CSOAI Compliance Passport and every Layer-0 governed action (the SIGIL chain).
